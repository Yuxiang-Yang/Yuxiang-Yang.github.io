Dropdown 下拉菜单 ​

将动作或菜单折叠到下拉菜单中。

基础用法 ​

悬停在下拉菜单上以展开更多操作。

  • item1
  • item2
  • item3
  • item4
<script setup>
import Button from '@/components/Button/Button.vue'
import Dropdown from '@/components/Dropdown/Dropdown.vue'
import DropdownItem from '@/components/Dropdown/DropdownItem.vue'

const options = [
  { key: 1, label: 'item1' },
  { key: 2, label: 'item2', disabled: true },
  { key: 3, label: 'item3', divided: true },
  { key: 4, label: 'item4' },
]
</script>

<template>
  <div>
    <Dropdown
      placement="bottom"
      :menu-options="options"
      trigger="hover"
    >
      <Button type="primary">
        Hover 下拉菜单
      </Button>
      <template #dropdown>
        <DropdownItem v-for="item in options" :key="item.key" :disabled="item.disabled" :divided="item.divided">
          {{ item.label }}
        </DropdownItem>
      </template>
    </Dropdown>
  </div>
</template>

触发方式 ​

可以配置点击激活或者悬停激活。

将 trigger 属性设置为 click 即可， 默认为 hover 。

<script setup>
import Button from '@/components/Button/Button.vue'
import Dropdown from '@/components/Dropdown/Dropdown.vue'

const options = [
  { key: 1, label: 'item1' },
  { key: 2, label: 'item2', disabled: true },
  { key: 3, label: 'item3', divided: true },
  { key: 4, label: 'item4' },
]
</script>

<template>

</template>

手动打开关闭 ​

将 manual 属性设置为 true 即可， 然后可以使用实例上面的 show 和 hide 方法打开关闭下拉菜单。

将 trigger 属性设置为 click 即可， 默认为 hover 。

  • item1
  • item2
  • item3
  • item4


<script setup>
import Button from '@/components/Button/Button.vue'
import Dropdown from '@/components/Dropdown/Dropdown.vue'
import DropdownItem from '@/components/Dropdown/DropdownItem.vue'
import { useTemplateRef } from 'vue'

const options = [
  { key: 1, label: 'item1' },
  { key: 2, label: 'item2', disabled: true },
  { key: 3, label: 'item3', divided: true },
  { key: 4, label: 'item4' },
]

const dropdownRef = useTemplateRef('dropdown')
function open() {
  dropdownRef.value?.show()
}
function close() {
  dropdownRef.value?.hide()
}
</script>

<template>
  <div>
    <Dropdown
      ref="dropdown"
      placement="bottom"
      :menu-options="options"
      manual
    >
      <Button type="primary">
        菜单容器
      </Button>
      <template #dropdown>
        <DropdownItem v-for="item in options" :key="item.key" :disabled="item.disabled" :divided="item.divided">
          {{ item.label }}
        </DropdownItem>
      </template>
    </Dropdown>
    <br /><br />
    <Button type="primary" @click="open">
      点击手动触发显示
    </Button>
    <Button type="danger" @click="close">
      点击手动触发隐藏
    </Button>
  </div>
</template>

属性名说明类型默认值

placement菜单位置enum-'top' | 'top-start' | 'top-end' | 'bottom' | 'bottom-start' | 'bottom-end' | 'left' | 'left-start' | 'left-end' | 'right' | 'right-start' | 'right-end'bottom
popper-optionspopper.js 参数object请参考 popper.js 文档—
show-after在触发后多久显示内容，单位毫秒number0
hide-after延迟关闭，单位毫秒number200
trigger如何触发 Tooltip'hover' | 'click' | 'focus' | 'contextmenu'—
manual是否开启手动触发模式booleanfalse
transition动画名称string—
hideAfterClick点击以后是否自动隐藏菜单booleantrue

Dropdown MenuOption ​

名称说明类型默认值
label菜单展示标签'string' | 'vNode'
key菜单选项 Key'string' | 'number'
disabled是否禁用booleanfalse
divided是否显示分隔符booleanfalse

Dropdown Events ​

事件名说明类型
visible-change当 tooltip 展示/隐藏时被触发boolean
select当选择到某一个选项的时候被触发MenuOption
